Cifra Caesaris — a fully interactive cipher tool with Roman aesthetic. Here's what's inside:
Three modes — Encode, Decode, and Compare (side-by-side both emperors)
Emperor selector — click Julius Caesar (Shift +3) or Augustus Caesar (Shift +1) to switch active cipher, with historical context for each
Live encoding — results update automatically as you type, no need to press the button
Compare mode — encodes your text with both ciphers simultaneously and shows a word-by-word comparison table
Custom shift — experiment with any shift value (1–25) beyond the two historical ones
Alphabet maps — visual A→D (Julius) and A→B (Augustus) shift charts for all 26 letters
Click to copy — tap any result to copy it to clipboard
